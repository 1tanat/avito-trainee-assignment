# Система управления объявлениями для модерации

Веб-приложение для модерации объявлений на платформе Авито.

## Технологии

- React 19
- TypeScript
- React Router DOM
- React Query
- Recharts
- Vite

## Требования

- Node.js v20+
- npm или yarn

## Установка и запуск

### 1. Запуск сервера

```bash
cd ../tech-int3-server
npm install
npm start
```

### 2. Запуск клиента

```bash
cd ../tech-client
npm install
npm run dev
```

## Функциональность

### Главная страница (/list)

- Отображение списка объявлений в виде карточек
- Фильтрация по:
  - Статусу
  - Категории
  - Диапазону цен
  - Поиск по названию или описанию
- Сортировка по:
  - Дате создания
  - Цене
  - Приоритету
- Пагинация

### Страница детального просмотра (/item/:id)

- Галерея изображений
- Полная информация об объявлении
- Характеристики товара
- Информация о продавце
- История модерации
- Панель действий модератора:
  - Одобрить
  - Отклонить
  - Вернуть на доработку
- Навигация между объявлениями

### Страница статистики (/stats)

- Общая статистика:
  - Количество проверенных объявлений
  - Процент одобренных, отклоненных, на доработке
  - Среднее время проверки
- Графики:
  - Активность по дням
  - Распределение решений
  - Распределение по категориям
- Фильтр по периоду:
  - Сегодня
  - Последние 7 дней
  - Последние 30 дней

## Необязательные технологии

- **TypeScript**: Типизация
- **React Query**: Упрощает работку с сервером
- **Recharts**: Графики
- **Vite**: Быстрая сборка
- **Prettier**: Форматирование кода
- **ESLint** - kbynth
